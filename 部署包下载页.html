🚀 部署指南

  1. 在 IDE 右侧「文件」面板中找到 public 文件夹
  2. 右键点击 public → 「下载」或「压缩」
  3. 上传到任一静态托管平台：
    • Vercel - vercel.com (免费)
    • Netlify - netlify.com (免费)
    • GitHub Pages - github.com (免费)
    • Cloudflare Pages - pages.cloudflare.com (免费)
💡 提示：以上文件可直接放入任何静态服务器，无需额外配置。
