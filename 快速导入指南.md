# 快速导入PDF题库 - 分步指南

## 🚀 快速开始（3分钟完成）

### 步骤1：准备PDF文件

1. 在 `c:\Users\Administrator\CodeBuddy\20260426000922\` 创建 `pdfs` 文件夹
2. 将你的IMA知识库PDF文件复制到 `pdfs` 文件夹中

示例：
```
pdfs\
  ├── IMA知识库-会计.pdf
  ├── IMA知识库-审计.pdf
  └── IMA知识库-税法.pdf
```

### 步骤2：运行一键脚本

**方法A：使用批处理脚本（最简单）**

双击运行：`install-and-parse.bat`

脚本会自动：
- ✅ 检查Node.js
- ✅ 安装必要依赖
- ✅ 检查PDF文件
- ✅ 启动解析工具

**方法B：手动执行**

打开命令行（cmd或PowerShell），执行：

```bash
cd c:\Users\Administrator\CodeBuddy\20260426000922

# 安装依赖
npm install pdf-parse

# 运行解析工具
node pdf-parser.js
```

### 步骤3：选择解析方式

解析工具会提供3种方式：

#### 方式1：手动输入（最准确）⭐推荐

- 适合题目数量较少（<100题）
- 可以逐题输入，确保准确性
- 支持批量连续输入

**操作流程：**
1. 输入科目名称（如：会计）
2. 输入章节（如：第一章 总论）
3. 输入题目类型（单选题/多选题/计算题）
4. 输入难度（1-3）
5. 输入题目内容
6. 输入选项A、B、C、D（E可选）
7. 输入正确答案（如：A 或 ABC）
8. 输入答案解析
9. 选择是否继续添加

#### 方式2：AI辅助解析（需要API）

- 自动解析PDF内容
- 需要配置AI API（如OpenAI）
- 适合大量题目

**注意：** 此功能需要额外配置，默认未启用

#### 方式3：查看文本后手动整理

- 程序会提取PDF文本保存到 `pdf-text-output.txt`
- 你可以查看文本，手动整理成JSON格式
- 然后使用导入工具导入

### 步骤4：导入题目到系统

解析完成后，题目会保存到 `imported-questions.json`

运行导入工具：

```bash
node import-to-database.js
```

按提示操作：
1. 确认题目数量
2. 选择导入方式（JSON文件 或 SQLite数据库）
3. 等待导入完成

### 步骤5：验证导入结果

启动服务器：

```bash
node server-simple.js
```

访问：`http://localhost:3000`

测试：
1. 点击首页的某个科目卡片
2. 检查题目是否正确显示
3. 尝试答题，查看答案解析

## 🔧 常见问题解决

### Q1: pdf-parse安装失败

**错误信息：**
```
npm ERR! code EPERM
npm ERR! syscall rename
```

**解决方法：**

以管理员身份运行命令提示符，然后执行：

```bash
npm install pdf-parse --force
```

或使用国内镜像：

```bash
npm install pdf-parse --registry=https://registry.npmmirror.com
```

### Q2: PDF解析出来是乱码

**原因：** PDF可能是扫描版或加密的

**解决方法：**

1. 使用OCR工具识别PDF（如Adobe Acrobat、ABBYY FineReader）
2. 将识别后的文本手动整理成JSON
3. 使用导入工具导入

### Q3: 题目格式不匹配

**原因：** PDF格式多样，自动解析无法匹配所有格式

**解决方法：**

1. 使用手动输入模式
2. 或修改 `pdf-parser.js` 中的正则表达式，适配你的PDF格式

### Q4: 导入后题目不显示

**检查方法：**

1. 检查 `data.json` 文件是否生成
2. 查看 `data.json` 中是否有题目数据
3. 重启服务器（`node server-simple.js`）
4. 清除浏览器缓存，重新访问

## 📊 批量导入 - 高级技巧

### 从Excel批量导入

如果你的题目在Excel中：

1. **准备Excel文件**

| subject | chapter | type | difficulty | question | option_a | option_b | option_c | option_d | correct_answer | explanation |
|---------|---------|------|-----------|----------|----------|----------|----------|----------|----------------|-------------|
| 会计 | 第一章 | 单选题 | 1 | 题目1 | A选项 | B选项 | C选项 | D选项 | A | 解析1 |
| 会计 | 第一章 | 多选题 | 2 | 题目2 | A选项 | B选项 | C选项 | D选项 | ABC | 解析2 |

2. **转换为JSON**

使用在线工具或脚本将Excel转为JSON：
- 在线工具：https://www.convertcsv.com/csv-to-json.htm
- 或手动整理成JSON格式

3. **导入系统**

```bash
node import-to-database.js
```

### 使用AI批量整理

如果你有PDF的文本版本（`pdf-text-output.txt`），可以使用AI帮助整理：

1. 将文本复制到ChatGPT/Claude等AI工具
2. 提示词示例：

```
请将以下CPA考试题目文本整理成JSON格式，每个题目包含：
- subject: 科目名称
- chapter: 章节名称
- type: 题目类型（单选题/多选题）
- question: 题目内容
- option_a, option_b, option_c, option_d: 选项内容
- correct_answer: 正确答案
- explanation: 答案解析

文本内容：
[粘贴你的题目文本]
```

3. 将AI生成的JSON保存到 `imported-questions.json`
4. 运行导入工具
