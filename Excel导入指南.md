# Excel批量导入指南

## 快速开始

### 方法一：使用批处理脚本（推荐）
```
双击运行: run-excel-import.bat
```

### 方法二：手动运行
```bash
# 1. 安装依赖
npm install xlsx

# 2. 运行导入
node excel-import.js
```

## Excel模板说明

### 模板文件位置
请使用 `题目导入模板-Excel格式.csv`，此文件可以直接在Excel中打开。

### Excel列说明

| 列名 | 必填 | 说明 | 示例 |
|------|------|------|------|
| 科目 | ✅ | 考试科目名称 | 会计、审计、税法、财务成本管理、经济法、公司战略与风险管理 |
| 章节 | ✅ | 章节或知识点 | 第一章 总论、第二章 存货 |
| 题目类型 | ✅ | 题目类型 | 单选题、多选题、计算题、简答题、综合题、判断题、案例分析题 |
| 难度 | ✅ | 难度等级 1-3 | 1（简单）、2（中等）、3（困难） |
| 题目内容 | ✅ | 题目文字内容 | 下列关于会计基本假设... |
| 选项A | 视题型 | A选项内容 | 会计主体确定了... |
| 选项B | 视题型 | B选项内容 | 持续经营明确了... |
| 选项C | 视题型 | C选项内容（可选） | 会计分期是... |
| 选项D | 视题型 | D选项内容（可选） | 货币计量为... |
| 选项E | 视题型 | E选项内容（可选） | 确认计量和报告的前提 |
| 正确答案 | ✅ | 正确答案字母 | A、ABC、ABD |
| 解析 | ❌ | 答案解析 | 会计主体确定了会计核算的空间范围... |

### 题目类型说明
- **单选题**: 4个选项，选1个正确答案
- **多选题**: 4-5个选项，选2个及以上正确答案
- **计算题**: 可能包含选项
- **简答题**: 无选项，直接写答案
- **判断题**: 正确答案填写 √ 或 ×
- **综合题**: 大题包含多个小问

## 使用步骤

### Step 1: 准备Excel文件

1. 打开Excel，新建工作簿
2. 按以下顺序创建表头（第1行）：
   ```
   科目 | 章节 | 题目类型 | 难度 | 题目内容 | 选项A | 选项B | 选项C | 选项D | 选项E | 正确答案 | 解析
   ```
3. 从第2行开始填写题目数据
4. 保存为 `.xlsx` 格式（或使用提供的CSV模板）

### Step 2: 放置文件

将Excel文件放入题库系统根目录，命名为：
- `题目导入模板.xlsx`（推荐）
- `题库.xlsx`
- `questions.xlsx`
- `import.xlsx`
- `data.xlsx`

### Step 3: 运行导入

双击 `run-excel-import.bat`，按提示操作。

## 常见问题

### Q1: 导入时提示"缺少xlsx依赖"
**解决**: 运行 `npm install xlsx`

### Q2: 导入后题目显示乱码
**解决**: 确保Excel文件编码为UTF-8

### Q3: 某些题目被跳过
**解决**: 程序自动跳过完全重复的题目（相同科目+相同题干）

### Q4: 如何查看已导入的题目？
**解决**: 查看 `data.json` 文件，或启动系统后在网页端查看

### Q5: 可以分批次导入吗？
**解决**: 可以，多次导入会自动合并，不会覆盖已有题目
